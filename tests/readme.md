## Test types

* **1** - Інструкція до тесту: нижче приведені чотири пари тверджень. З кожної пари виберіть те твердження, яке Вас характеризує, яке Вам більше підходить та запишіть його порядковий номер. У підсумку у Вас має вийти комбінація з чотирьох цифр. Нижче, до кожної комбінації, надається інтерпретація типів особистості та рекомендації щодо вибору можливих варіантів професій.
